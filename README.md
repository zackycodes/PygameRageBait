# Pygame 2D Rage Bait

## How to Play
- wasd keys
- you are the blue cube
- enemy is the red cube
- touch enemy cube to +1 score
- there's yellow and purple cubes everywhere
- beware of em.

## Rage Bait elements
- Sudden speed boosts
- Controls inversion
